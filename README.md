# Szűcs Noel Gergő - Minimalista React oldal

Egyszerű, modern, egyoldalas személyes weboldal React + TypeScript + Vite alapon.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Gyors indítás

1. Telepitsd a Node.js LTS verziot.
2. Telepitsd a fuggosegeket:

```bash
npm install
```

## Discord

<a href="https://discord.com/users/1499037894386647133"><img src="https://lanyard.cnrad.dev/api/1499037894386647133?showDisplayName=true" /></a>

3. Inditsd el a fejlesztoi szervert:

```bash
npm run dev
```

4. Nyisd meg a terminalban megjeleno lokal URL-t (altalaban: `http://localhost:5173`).

## Build es preview

Production build:

```bash
npm run build
```

Build elozetes megtekintese lokalisan:

```bash
npm run preview
```

## Projekt struktura

Fo fajlok:

- `index.html` - Alap HTML sablon
- `src/main.tsx` - Alkalmazas belepesi pontja
- `src/App.tsx` - Fo oldalkomponens
- `src/index.css` - Globalis stilusok
- `src/components/` - Ujrafelhasznalhato UI komponensek

## Elerheto npm scriptek

- `npm run dev` - Fejlesztoi szerver inditasa
- `npm run build` - Production build keszitese
- `npm run preview` - Build futtatasa lokalis preview modban

## Megjegyzes

Ha a fejlesztoi szerver nem indul, ellenorizd, hogy a Node.js verzio megfelelo-e, majd futtasd ujra:

```bash
npm install
```
